# 🤖 Jarvis AI Assistant

An intelligent AI-powered virtual assistant developed as my final-year diploma project. Jarvis combines voice interaction, online and offline AI models, and remote control features to provide a smart, secure, and productivity-focused assistant.

---

## 🚀 Features

* 🎤 Voice command recognition
* 🔊 Text-to-Speech (TTS) responses
* 🧠 Conversational AI using Google Gemini
* 💻 Offline AI support with a local LLM
* 💾 Memory system for personalized interactions
* 📱 Telegram bot integration for remote access
* 📂 File management (send, receive, organize files)
* 🖥️ System automation through voice commands
* 🔒 Safe Mode to prevent unsafe operations
* ⚡ Fast and responsive user experience

---

## 🛠️ Technologies Used

* Python
* Google Gemini API
* Speech Recognition
* Text-to-Speech (TTS)
* Local LLM
* Telegram Bot API
* JSON / SQLite (Memory Storage)
* Git & GitHub

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/jarvis-ai.git
```

2. Navigate to the project

```bash
cd jarvis-ai
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Configure your API keys.

5. Run the assistant

```bash
python main.py
```
